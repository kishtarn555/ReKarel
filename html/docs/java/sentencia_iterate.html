Iterate

La sentencia iterate permite a Karel repetir una instrucción o bloque múltiples veces.

A continuación se muestra la estructura de la sentencia iterate. Lo que hace, es repetir una instrucción el número de veces indicado.

iterate (NÚMERO)
INSTRUCCION

Por ejemplo, un código que avance 14 veces se vería de la siguiente manera

iterate (14) {
move();
}

La sentencia iterate también acepta parámetros en vez de un número. Puedes leer más al respecto sobre los parámetros aquí
