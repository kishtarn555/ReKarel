Primeros pasos

Puedes ver un curso corto para aprender a usar el editor, a programar en ReKarel y resolver problemas básicos

Primero, necesitas elegir entre ReKarel Java y ReKarel Pascal, estos son los dos lenguajes de programación que se ofrecen en ReKarel

  • ReKarel Pascal: Lenguaje de programación basado en Pascal (1970), el lenguaje se encuentra en español
  • ReKarel Java: Lenguaje de programación basado en Java (1995), el lenguaje se encuentra en inglés
