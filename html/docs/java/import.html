Import

Desde ReKarel 2.0.0

ReKarel es capar de importar librerías internas para incrementar las funciones que Karel puede realizar.

Para importar una librería, en la parte más superior del código se debe escribir import seguido del nombre de la librería, el cual tiene un formato de namespace.paquete

Veamos un ejemplo:

import rekarel.globals;
class program {
    program () {
        turnoff();
    }
}

Librerías disponibles

Actualmente la única librería disponible es rekarel.globales
