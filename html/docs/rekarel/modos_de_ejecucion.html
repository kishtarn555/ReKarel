Modos de ejecución

ReKarel IDE ofrece multiples modos de ejecución.

Modo normal

Este es el modo por defecto, en este modo todos los sistemas de ReKarel están disponibles.

Modo rápido

En este modo, ReKarel desactiva algunos sistemas para priorizar la ejecución de código de ReKarel.

La pila de llamadas deja de actualizarse, este es uno de los sistemas más lentos de ReKarel, la pila queda congelada a como quedo cuando se activo este modo.

Este modo es activado automáticamente cuando se supera el límite de instrucciones establecido en la configuración y se desactiva al reiniciar el mundo.
