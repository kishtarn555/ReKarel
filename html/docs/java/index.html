Documentación de ReKarel Java

ReKarel Java es un lenguaje de programación utilizado para controlar a Karel el Robot. La sintaxis del lenguaje se basa en Java.

Versión

Esta documentación explica la version de ReKarel Java 2, que agrega características nuevas a comparación de Karel Java, puedes leer más aquí.

Mientras que la mayoría de códigos de Karel Java compilan en ReKarel-Java-2, no todos lo hacen por nuevas palabras clave.

Tutorial

Esta documentación no esta diseñada con el propósito explicito de enseñar a usar Karel o ReKarel Java, sino que está diseñada para aclarar dudas del lenguaje y usarlo como una referencia para consultas breves. Aunque es cierto que se puede utilizar con este propósito, recomendamos utilizar un tutorial o curso propio.
