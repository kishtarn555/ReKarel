Documentación de ReKarel Pascal

ReKarel Pascal es un lenguaje de programación utilizado para controlar a Karel el Robot. La sintaxis del lenguaje se basa en Pascal.

Versión

Esta documentación explica la version de ReKarel Pascal @@languageVersion, que agrega características nuevas a comparación de Karel Pascal, puedes leer más aquí.

Mientras que la mayoría de códigos de Karel Pascal 1 compilan en ReKarel Pascal @@languageVersion, no todos lo hacen por nuevas palabras clave.

Tutorial

Esta documentación no esta diseñada con el propósito explicito de enseñar a usar Karel o ReKarel Pascal, sino que está diseñada para aclarar dudas del lenguaje y usarlo como una referencia para consultas breves. Aunque es cierto que se puede utilizar con este propósito, recomendamos utilizar un tutorial o curso propio.
