Introducción a la documentación

Bienvenido a la documentación de Karel. Un lugar donde poder leer rápidamente información de Karel Pascal.

Algo importante destacar es que esta documentación NO ES UN TUTORIAL DE KAREL. Ya que los temas no están ordenados para este motivo, si no que está diseñado para poder resolver dudas que se puedan tener del lenguaje.

Puedes ver un tutorial aquí

Karel pascal es un lenguaje que permite controlar a Karel utilizando una sintaxis basada en el lenguaje "Pascal", una gran parte de lo que aprendamos será útil también en ese lenguaje.
